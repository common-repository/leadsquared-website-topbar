== Description ==
Website Topbar helps you direct your website visitors to a web page that matters to you. You can use it to direct your visitors to your product purchase page, your webinar registration page, your newsletter sign-up page or any landing page that matters. It is a very easy to use plugin with complete configuration of the text and colors of the bar. You can even control how long you want to display it. The help documentation is on http://help.leadsquared.com/how-can-i-use-leadsquareds-website-top-bar-plugin-for-wordpress/

== Installation ==

In your WordPress Dashboard, from left panel select **Plugins->Add New**
then SELECT the zip file of **leadsquared-website-topbar.zip** hit INSTALL
After a few seconds there is an option for **Activate Plugin**
click the link of Activate Plugin. 

 == Frequently Asked Questions ==
Can I Install the plugin manually ?
Yes, you can. Extract the zip file of the "leadsquared-website-topbar.zip" then place the whole folder in the following location
"yousite\wp-content\plugins\".

Do I need WordPress newer than 3.0 ?
Ans - Yes, your WordPress version should be greater than 3.0

Can I use HTML to tweak the text and effects in the top bar!
Ans - Absolutely! For example, if you use <b>My Text</b>, it will help you make your text bold.

How do I uninstall?
Go to the Uninstall tab and then click the Uninstall button at the bottom of the page. You'll be sent to the WordPress Plugins page. Now deactivate and uninstall the Plugin

How do I hide the Top Bar?
Just go to the configuration page, Click on "Hide", just above the Save button, and then click on Save. The top bar won't be displayed till it is "Shown" again.
